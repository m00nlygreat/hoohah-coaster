## Hoohah-coaster

1. 심호흡을 도와주는 비주얼 가이드 앱입니다.
2. 먼저 가로축이 시간, 세로축이 숨의 양인 2D Line Graph를 그립니다. 이 그래프는 배경에서 횡스크롤 되며 현재 시간이 가운데에 고정되어 시간의 흐름을 표현합니다.
3. 현재 시간을 나타내는 노란색 공을 Line Graph 위에 표시합니다. 이 공의 X 값은 고정이며, Y 값은 왼쪽으로 횡스크롤되는 Line Graph를 타고 오르내립니다.
4. 사용자는 공의 위 아래 움직임을 보고 숨을 마시거나 뱉게 됩니다.
5. 4초 마시기, 7초 멈추기, 8초 내쉬기, 2초 멈추기의 구성으로 무한 반복되는 가이드를 제작하세요.
6. 시각을 사로잡는 Rich한 애니메이션으로 만들어야 합니다.
7. Vanilla JS / 단일 HTML 파일로 구현
