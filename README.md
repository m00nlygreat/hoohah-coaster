# Hoohah-Coster

웹에서 호흡 리듬을 시각화하고, 각 단계의 수준·시간을 자유롭게 편집할 수 있는 실험용 데모입니다. 로컬 스토리지에 패턴과 이징 모드를 저장해 다시 방문해도 설정이 유지됩니다.

## 실행 방법
1. 이 디렉터리에서 index.html을 브라우저로 직접 열면 됩니다.
2. 더블클릭으로 전체화면 토글, 배경 클릭으로 설정 패널을 닫을 수 있습니다.

## 주요 기능
- **캔버스 시각화**: 호흡 레벨을 실시간 곡선과 중앙 포인트로 표현.
- **단계 편집**: 단계 추가/삭제, 드래그로 순서 변경, 슬라이더·숫자 입력으로 레벨(0~1)과 시간(초) 조정.
- **이징 모드**: Off(기본), Line, Screen 선택으로 움직임 느낌 변경.
- **저장**: Save 버튼으로 현재 패턴을 브라우저 로컬 스토리지에 저장.

## 로컬 데이터
- 키: reathPattern, reathEaseMode
- 브라우저 캐시를 지우면 저장된 패턴이 초기화됩니다.

## 참고
간단한 데모로, 모바일 및 데스크톱 모두에서 동작하지만 고해상도 디스플레이에서는 GPU 자원 사용이 늘어날 수 있습니다.
